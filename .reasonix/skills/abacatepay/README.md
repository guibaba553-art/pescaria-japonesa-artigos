# AbacatePay Skills 🥑

Conjunto de conhecimentos, regras e ferramentas especializadas para potencializar a integração com a **AbacatePay** através de **IAs** e automações.

<img src="https://res.cloudinary.com/dkok1obj5/image/upload/v1767631413/avo_clhmaf.png" width="100%" alt="AbacatePay Open Source"/>

## O que são SKILLS?

Skills são pacotes de contexto especializado que ensinam a **agentes de IA** (como **Cursor**, **Antigravity** e **Claude Code**) e desenvolvedores sobre as melhores práticas, regras de negócio e padrões técnicos do ecossistema AbacatePay. Elas funcionam como um "Manual de instruções inteligente".

## Para que servem?

</div>

- **Contexto Preciso:** Fornece à IA informações atualizadas sobre APIs, SDKs e Webhooks.
- **Padronização:** Garante que o código gerado siga as convenções da AbacatePay.
- **Aceleração:** Reduz o tempo de leitura de documentação extensa através de regras diretas.
- **Exemplos Práticos:** Oferece implementações de referência em TypeScript e Go.

<div align="center">

## Como Instalar

As skills são consumidas principalmente por **IDEs** e **agentes de IA** que suportam leitura de contexto local

</div>

1. **Clonagem:** Clone este repositório em sua máquina.
2. **Contexto da IA:** Adicione a pasta `skills/` ao contexto do seu editor (Exemplo, `.cursorrules`).
3. **CLI:** Algumas skills são integradas diretamente na [AbacatePay CLI](https://docs.abacatepay.com/cli).

<div align="center">

## Como Usar

A estrutura é dividida para facilitar a descoberta

</div>

- **[`tests/`](./tests):** Suítes de teste para validar as integrações.
- **[`tools/`](./tools):** Documentação técnica de ferramentas de apoio e SDKs.
- **[`examples/`](./examples):** Implementações reais e snippets prontos para uso.
- **[`rules/`](./rules):** Regras específicas por linguagem (TS/Go) para cada módulo (Checkout, Subscrições, etc).
