# No-Code / Low-Code Integration

Tips for using AbacatePay without needing to code.

## Overview

AbacatePay supports no-code and low-code integrations through various tools and platforms. This allows non-technical users to accept payments easily.

## Available Options

### 1. Dashboard Integration
- Use the AbacatePay dashboard for manual payment creation
- Generate payment links without code
- Monitor transactions through the web interface

### 2. Third-Party Platforms
- Integrate with no-code platforms like Zapier, Make, or Bubble
- Use pre-built connectors and workflows
- Automate payment processes with visual workflows

### 3. CMS Plugins
- WordPress plugins for easy e-commerce integration
- Shopify apps for seamless payment processing
- Other CMS integrations for content management systems

### 4. E-commerce Platforms
- WooCommerce integration
- Magento connectors
- PrestaShop modules

## Getting Started

1. **Sign up** for AbacatePay account
2. **Choose your platform** (WordPress, Shopify, etc.)
3. **Install the plugin/app** from the platform's marketplace
4. **Configure API keys** in the plugin settings
5. **Test payments** in dev mode

## Benefits

- No programming knowledge required
- Quick setup and deployment
- Visual configuration interfaces
- Community support and templates

## Limitations

- Less customization than custom code
- Dependent on platform updates
- May have additional fees from third-party platforms
